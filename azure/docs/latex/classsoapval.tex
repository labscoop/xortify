\subsection{Detailed Description}
For creating serializable abstractions of native P\-H\-P types. This class allows element name/namespace, X\-S\-D type, and X\-M\-L attributes to be associated with a value. This is extremely useful when W\-S\-D\-L is not used, but is also useful when W\-S\-D\-L is used with polymorphic types, including xsd\-:any\-Type and user-\/defined types.
