Help: Xortify

Description

Xortify is a client for the protector honeypot cloud for protection from Spam, Harvesting & Hacker. This client can work in conjunction with protector or standalone and utilises honeypots on the web once you have signed up!

Install/uninstall

No special measures necessary, follow the standard installation process - extract the /htdocs folder into the XOOPS Root Path. Install the module through Admin -> System Module -> Modules. If you need detailed instructions on how to install a module, please see the XOOPS Operations Manual.

Once you have installed Xortify you will need to sign up a username and password for the cloud if you don't have a username and password, you can have as many of these as you need; it will set the details in the configuration for the module.
