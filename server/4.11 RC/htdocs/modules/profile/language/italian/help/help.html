Help: Profile

Descrizione

Il modulo Profile fornisce un profilo agli utenti e le fasi di registrazione, degli stessi utenti nella fase di registrazione. Insieme a brevi descrizioni e, opzionalmente, gestione dei contenuti degli stessi utenti.

Installa/disinstalla

Nessuna misura particolare, seguire il processo di installazione standard ed estrarre la cartella profile nella directory ../modules. Installare il modulo attraverso Amministrazione -> Modulo di sistema -> Moduli.
Istruzioni dettagliate per l'installazione dei moduli sono disponibili nella XOOPS Operations Manual
